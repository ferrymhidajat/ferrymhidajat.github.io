HTML - DAY 1

Minggu pertama, hari pertama kita membahas dan mempelajari apa sih itu HTML. Tugas yang kita harus kerjakan adalah berupa tutorial interaktif di codeacademy yang cukup panjang. Di tutorial ini, kita diajarkan bagaimana sih memfungsikan dan memperkerjakan HTML dalam sebuah website. Ternyata, HTML adalah bagian pembangun dasar sebuah website. Singkatnya, dia merupakan kerangkanya si website. Saat ini versi resmi paling update disebut HTML5.

Tags dan Attributes

  • !doctype html digunakan untuk mendeklarasi tipe file menjadi HTML
  • head./head memuat meta data berupa title./title yang menjadi judul dokumen dan ./meta, data dan informasi sebuah halaman
  • h1-h6 merupakan heading yang biasanya digunakan sebagai judul sebuah paragraf
  • body./body akan menunjukkan konten halaman, yang berisi div./div yang digunakan sebagai pembagi dan pemisah
  • p./p digunakan untuk membuat berupa paragraf

Bentuk dari kodingan HTML ini memiliki kerangka dan struktur yang relatif terus digunakan dalam semua jenis dan variasi website.

CSS - DAY 2

Minggu pertama, hari kedua kita memulai untuk mempelajari apa sih itu CSS. Tugas berupa tutorial interaktif di codeacademy juga. Di tutorial ini, kita diajarkan bagaimana sih memfungsikan dan memperkerjakan HTML dalam sebuah website. Ternyata, HTML adalah bagian pembangun dasar sebuah website. Singkatnya, dia merupakan kerangkanya si website. Saat ini versi resmi paling update disebut HTML5. CSS digunakan untuk mengedit penampilan website untuk dispesifikasi untuk bagaimana caranya untuk membuat website yang menarik untuk dilihat. Cascading Style Sheet terdiri dari beberapa hal seperti selektor, deklarasi, Properti dan Nilai. Seperti pada HTML, PHP dan bahasa pemrograman lainnya, CSS juga memiliki aturan yang menulis itu sendiri.

CSS digunakan sebagai atribut/elemen/bagian dari tag HTML. Didalam CSS juga terdapat beberapa versi yang terus diupdate dan diperbaharui seperti HTML2 sampai yang sekarang HTML 5, CSS3 adalah versi terbaru yang merupakan versi terbaru dari CSS 2.1 dengan penambahan beberapa fitur yang ada. Anda dapat mempelajari dan melihat pembaharuan atau perkembangannya di website resminya

Contoh dari penggunaan CSS adalah menghubungkan tag HTML dengan CSS. Misalnya *p* digunakan untuk diubah bentuk dari jenis fontnya menggunakan *font-family*, color digunakan untuk mengubah warna font menggunakan *color*, *font-size* digunakan untuk mengatur besar kecilnya sebuah font size yang digunakan untuk mengubah tampilannya. Biasanya ada margin yang digunakan dalam perubahan letak suatu element HTML yang dapat dilakukan dalam CSS.

GIT - DAY 3&4

Dihari ketiga dan keempat kita mencoba membuat account GitHub dan belajar apa fungsinya dari GitHub itu sendiri. GitHub merupakan platform untuk para programmer yang menjalankan suatu kolaborasi atau juga dapat digunakan seorang programmer yang ingin memiliki archive online hasil kodingannya. Git main focusnya adalah version control, yaitu mengupdate kodingan dari sebuah team dengan upaya agar tidak ada konflik dari sebuah koding dari secara keseluruhan. Artinya dari kodingan yang diedit dari semua programmer tidak ada versi yang memiliki kodingan yang bertentangan atas satu sama lain, file dirubah secara bergantian.

Dalam penggunaan GitHub di lakukan pengaturan penempatan dan command-command yang dapat dijalankan terminal gitbash, seperti *mkdir xavier-fox* yang berarti buat folder berjudul xavier-fox. Lalu ada *touch index.html* yang berfungsi untuk membuat extension file atau file. *Open -a Line* open dapat digunakan sebagai command untuk membuka software, disini contoh softwarenya adalah Line Messenger. *Cp_(namafile)(namafilebaru) digunakan untuk mengcopy dan mengganti nama sebuah file.

Adapun command yang kita bisa input dalam *mv_(nama-file)_example, examplenya ini folder yang ingin filenya itu dipindahkan. *rm_file* rm digunakan untuk menghapus file, file yang dihapus di rm hilang, seperti di ctrl+alt+del dan tidak bisa diambil kembali. *rm_rf_folder* digunakan untuk menghapus delete folder. *mkdir_test*&&*cd test*&&*touch index.html* adalah contoh penggunaan beberapa command sekaligus. Disini kita pertama membuat folder bernama test, lalu membuka folder, membuat file dengan nama index dan jenis ekstension html.
